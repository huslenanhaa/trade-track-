# Trade Track Pro

Trade Track Pro is a trading journal and analytics web application built with React, Vite, Express, and Supabase. It helps traders record trades, import trade history, review performance, test strategies, and track trading progress from one dashboard.

## Features

- Trading dashboard with performance summary cards
- Trade journal pages for reviewing and managing trades
- CSV trade import support
- Backtesting and replay tools
- Market data and TradingView chart integration
- Login, reset password, and account settings pages
- Responsive interface built with Tailwind CSS

## Tech Stack

- React
- Vite
- Tailwind CSS
- Express
- Supabase
- Recharts
- Three.js

## Getting Started

Install dependencies:

npm install

Run the development app:

npm run dev

Run the backend server:


npm run server


Build for production:

npm run build

 Notes

The `node_modules`, `dist`, build output, and environment files are intentionally ignored from Git. After cloning the repository, run `npm install` before starting the project.
